# 作品說明

- 查詢全台各縣市景點、活動、美食
- 使用 Vue 3 Composition API
- 介接 TDX 運輸資料 API

# 系統說明

## Project setup
npm install

### Compiles and hot-reloads for development
npm run dev

### Compiles and minifies for production
npm run build
